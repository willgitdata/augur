# Migrating

`0.1.0` is the first public release of Augur — there are no prior
published versions to migrate from. If you're new to the package,
start with the [README](README.md) and [EXAMPLES.md](EXAMPLES.md).

This file will populate once subsequent versions ship. Augur is pre-1.0,
so breaking changes can land on `0.x` minor bumps; when they do, each
upgrade gets its own self-contained section here covering the old
shape, the new shape, and the smallest diff that gets you across.

For the per-release additions and changes (breaking and non-breaking),
see [CHANGELOG.md](CHANGELOG.md).
