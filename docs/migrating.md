# Migrating

`0.1.0` was the first public release of Augur but shipped with an empty `dist/` and was deprecated on npm. `0.1.1` is the same feature set packaged correctly. There are no prior published versions to migrate from. If you're new to the package, start with the [README](../README.md) and [examples.md](./examples.md).

This file will populate once subsequent versions ship. Augur is pre-1.0, so breaking changes can land on `0.x` minor bumps; when they do, each upgrade gets its own section here covering the old shape, the new shape, and the smallest diff that gets you across.

For per-release additions and changes (breaking and non-breaking), see [CHANGELOG.md](../CHANGELOG.md).
